
**Markdown** is a lightweight markup language designed to be easy to read and write.
MD files use simple plain text with minimal formatting symbols.
Commonly used for creating documentation, README files, and basic content on websites.
Supports basic text formatting such as headings, lists, links, and code blocks.
Easy to learn and widely supported by various text editors and platforms.
Not suitable for embedding complex components or dynamic content.

Example of Markdown syntax:

```markdown
# Heading 1

## Heading 2

- List item 1
- List item 2

[Link](https://example.com)
```
