Static Sites​

Ipein Notes​

Ipein Notes est un site web que j'ai créé avec mkDocs, où je partage des documents de cours avec mes étudiants (supports de cours, travaux dirigés, travaux pratiques, corrigés).

  • Lien : Ipein Notes
  • Dépôt GitHub : Dépôt GitHub

Outils et Technologies Utilisés :

  • mkDocs: Un framework pour la création de sites web statiques.
  • Markdown: Un langage de balisage léger utilisé pour formater du texte de manière structurée et lisible par l'homme, particulièrement pour la création de contenu web.
  • Typora: Un éditeur de texte Markdown avec une interface utilisateur intuitive.
  • VSCode (Visual Studio Code): Un environnement de développement intégré (EDI) open source offrant une grande extensibilité et des outils puissants pour le développement de logiciels.
  • Python: Un langage de programmation polyvalent utilisé pour écrire des scripts et des applications.
  • Git: Un système de gestion de version pour les projets logiciels.
  • GitHub: Une plateforme en ligne qui utilise Git pour le stockage et la gestion de projets logiciels.
